# HR UI 스타일 추천 시스템
고객사 브랜드 정보를 입력하면 Gemini AI가 HR 플랫폼에 적합한 UI 컬러 시스템을 자동 추천합니다.

## 개발 배경
HR 솔루션 구축 프로젝트마다 고객사 CI에 맞춰 컬러를 수작업으로 변경하는 반복 업무를 AI로 자동화하고자 개발했습니다.

## 기술 스택
- Vue.js 3
- Gemini API (Google AI)
- Vite

## 실행 방법
1. `.env` 파일 생성 후 API 키 입력
2. `npm install`
3. `npm run dev`

## 주요 기능
- 회사명, 메인 컬러, 산업군, UI 스타일 방향 입력
- Gemini AI가 8가지 UI 컬러 시스템 자동 추천
- 컬러 박스로 시각적 결과 확인
- 추천 이유 제공
